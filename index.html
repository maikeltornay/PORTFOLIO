Estos son los lenguajes de programacion que estoy adquiriendo

  • Git

    Git es un sistema de control de versiones. Es como una "máquina del tiempo" para tu código: te permite guardar un historial de todos los cambios que haces en tus archivos, volver a versiones anteriores si algo se rompe y trabajar en varias funciones a la vez sin mezclar el código principal.

  • Html

    HTML (HyperText Markup Language) es el esqueleto de una web. No es un lenguaje de programación, sino un lenguaje de marcado que utiliza etiquetas (como h1, p o div) para estructurar el contenido: decirle al navegador qué es un título, qué es un párrafo y dónde va una imagen.

  • Css

    CSS (Cascading Style Sheets) es el traje y el maquillaje de tu web. Es el lenguaje que usamos para dar estilo al esqueleto de HTML. Con CSS decides los colores pastel de tu logo, el tamaño de las letras, la posición de los iconos y que tu web se vea perfecta tanto en el móvil como en el ordenador.

  • Javascript

    JavaScript es el cerebro de tu web. Es el lenguaje de programación que permite que tu sitio haga cosas: desde que un botón cambie de color al pulsarlo, hasta validar un formulario de contacto o cargar animaciones. Si el HTML es el esqueleto y el CSS la ropa, JavaScript es el movimiento y la interacción.

  • Base de datos

    Las Bases de Datos son el almacén o el archivo de tu web. Es donde se guarda toda la información de forma organizada y permanente: desde los nombres de tus usuarios hasta el contenido de tus posts. Mientras que JavaScript gestiona lo que pasa en el momento, la base de datos se encarga de que la información no se pierda al cerrar el navegador.

  • Node js

    Node.js es el motor que permite que JavaScript funcione fuera del navegador. Es el entorno que utilizas para construir el servidor de tu aplicación. Gracias a Node.js, puedes usar el mismo lenguaje con el que haces los efectos visuales (JavaScript) para gestionar bases de datos, crear usuarios y controlar toda la lógica interna de tu web.

  • React

    React es la fábrica de piezas (componentes) para tu interfaz. Es una biblioteca de JavaScript que te permite crear la web por partes reutilizables. En lugar de escribir toda la página de golpe, creas la "pieza" del botón, la del logo y la del menú por separado, y React se encarga de que se actualicen de forma rápida y eficiente cuando algo cambia.

  • Phyton

    Python es el multiusos de la programación. Es un lenguaje famoso por ser muy fácil de leer y escribir (se parece al inglés). Se usa para casi todo: desde crear el servidor de tu web (con Django o Flask) hasta analizar datos, crear Inteligencia Artificial o automatizar tareas aburridas. En un portfolio, Python dice que eres un desarrollador versátil.
